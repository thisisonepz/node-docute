## 工作单取消

1. 请求方式：POST；

2. URL：workOrders/cancelOrder

3. 请求示例：http://68.6.2.56:9999/workOrders/cancelOrder

4. 请求参数：
```$xslt
{
  "gzdid":"工作单id",
  "bz":"取消原因"
}
```
5. 响应
```$xslt
{
  {
  "data": {
     "jkds":[{
         "缴款单1","缴款单2"
     }],
```
